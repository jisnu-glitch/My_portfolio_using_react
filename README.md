# Jishnu Vijayan – Personal Portfolio

A professional portfolio website designed to showcase a modern identity, technical skills, and creative projects. This project features a cinematic dark aesthetic with seamless animations and a fully responsive layout.

---

## Overview

This portfolio serves as a centralized hub for my professional profile, highlighting key expertise in frontend development and UI/UX design. The application is built with a focus on high performance, smooth user transitions, and a premium visual experience.

---

## Core Features

- **Responsive Design**: Optimized for a consistent experience across mobile, tablet, and desktop devices.
- **Cinematic Aesthetic**: A refined dark theme with consistent typography and visual depth.
- **Interactive Components**: Including a modern Hero section, Skills matrix, Project gallery, and Experience timeline.
- **Micro-animations**: Smooth entry effects and interactive elements powered by ScrollReveal and Framer Motion.
- **Educational Context**: A dedicated section for academic background and language proficiency.

## Technology Stack

### Development Tools
- **React 19**: Modern component-based architecture.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS**: Utility-first CSS framework for rapid and responsive UI development.

### Libraries & Utilities
- **ScrollReveal**: For smooth scroll-based entrance animations.
- **Framer Motion**: Advanced motion components for complex transitions.
- **PostCSS & Autoprefixer**: For cross-browser CSS compatibility.

## Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server locally:
```bash
npm run dev
```

### Production Build

Create an optimized build for production:
```bash
npm run build
```
